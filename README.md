# Tank Parts  
### Beta  
  
This mod adds varoius breeches and shells.  
To use this mod you should install [SM-DLM](https://github.com/QuestionableM/SM-DLL-Injector/tree/v1.4#readme) first.  
SM-DLM isn't necessary but recommended.
